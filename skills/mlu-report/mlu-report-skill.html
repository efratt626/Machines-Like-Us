Claude Skill

MLU Report Builder

Generates a styled Excel QBR or Monthly report for Machines Like Us from platform analytics CSV exports — Apple, Spotify, and YouTube.

Apple Podcasts Spotify YouTube Excel Output

When to Use

Say any of the following — Claude will invoke this skill automatically:

"Generate the Q2 QBR" "Create the March monthly report" "Run the MLU report for Q3" "Build the report" "Make the April report" "Run the report for Q1 2026"

How It Works

1

Identify Inputs

Claude asks for (or infers) the root folder, period label (e.g. Q1 2026 or March 2026), and report type (QBR or Monthly).

2

Validate Folder Structure

Before running, the script checks that all required subfolders and files exist. Any missing items are reported clearly before anything runs.

3

Generate Charts & Build Workbook

The bundled Python script auto-discovers all CSVs via glob, renders four styled charts, and assembles the Excel workbook with data tables alongside each chart.

4

Save & Confirm

The finished workbook is saved to the root folder. Temporary chart images are deleted automatically. Claude reports the full output path.

Required Folder Structure

<root>/
  Apple/
    Apple Monthly Listeners/  ← one CSV per month
      MLU_ListenersbyEngagement_Jan2026.csv
      MLU_ListenersbyEngagement_Feb2026.csv
      MLU_ListenersbyEngagement_Mar2026.csv
    Apple Monthly Subscribers/  ← one CSV per month
      MLU_Jan26_Apple_MonthlySubs.csv
      MLU_Feb26_Apple_MonthlySubs.csv
      MLU_Mar26_Apple_MonthlySubs.csv
  Spotify/
    MLU_Q126_Spotify_Listeners.csv  ← single file
  YouTube/
    MLU_YouTube_Q126_AllViewers.csv  ← single file
    MLU_YouTube_Q126_Subscribers.csv  ← single file
Files are discovered via glob — you only need to drop new monthly CSVs into the correct subfolder. No code changes required for new months or quarters.
